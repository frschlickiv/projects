Breaking into Tech

So much Javascript

I started learning how to code in the spring 2021. It has been a cyclical learning process that has required me to use a number of different resources. Like anyone else, I started with freeCodeCamp. Then I moved onto codecademy and used a number of different fundamentals based courses to get the ball rolling. The vast majority of my coding experience is in Javascript. You can see some of my Javascript projects here.

My first major project is a tic-tac-toe command line game. (I know how original) The important aspect of this project is understanding pass by reference and mutating an object with a number of different functions.

Now I am working my way through more front-end skills with Colt Steele's Web Developer Bootcamp. I will continue Launch school when the time is right, but for now, I want to focus on front-end before I try to break into back-end development. Some other amazing resources I have used are Odin Project, CSX by Codesmith, App Academy Open, and codewars. I have a great deal of drive to break into this industry. I would like to join a coding bootcamp, but it's an expense that I don't feel ready to take on at the moment. Maybe when I can focus solely on web development it will be easier to consolidate my energy.
